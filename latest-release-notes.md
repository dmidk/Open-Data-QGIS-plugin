## Changes

 - The DMI open data API has changed url (https://www.dmi.dk/friedata/dokumentation/release-notes) and no longer requires api-key.
 - Settings page removed as api-key is no longer needed.

## Installation

Download the latest zip file build from the Assets section. Install it by going to Plugins -> Manage and install plugins -> Install from ZIP, and select the zip file you just downloaded.
